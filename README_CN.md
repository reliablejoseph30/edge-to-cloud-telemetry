# Edge-to-Cloud Telemetry System

## Overview

This project focuses on transmitting telemetry data from edge devices to the cloud for monitoring and analysis.

The goal is to simulate how real-world systems (such as drones or IoT devices) send data reliably to cloud infrastructure.

## Key Concepts

* Edge computing
* Cloud communication
* Telemetry data streaming
* System reliability
